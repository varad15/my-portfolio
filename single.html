Pune Municipal Corporation – Complaint Data Analysis

Role: Project Lead | Tech: Data Analysis, ML

Project Overview

Analyzed multi-year civic complaint data for the Pune Municipal Corporation to uncover trends, inefficiencies, and escalation patterns. The primary goal was to move from a reactive to a predictive system for handling citizen complaints. Our analysis identified key bottlenecks in the existing workflow.

The core objective was to propose an ML-driven workflow for automatic complaint categorization and routing, aiming to significantly improve response efficiency and citizen satisfaction.

Technical Approach
The initial phase involved extensive data cleaning and preprocessing of unstructured text data from thousands of complaints. We utilized Natural Language Processing (NLP) techniques to extract features and classify complaint types. The proposed system was designed around a YOLOv8 model (for image-based complaints like potholes) and a BERT-based text classifier.

Key Responsibilities

  • Led the project team, coordinating tasks and timelines.
  • Performed in-depth data analysis and visualization to identify trends.
  • Developed and trained the ML models for classification and routing.
  • Authored the final report and presented findings to stakeholders.
